# My Workflow

## Purpose

This is a starter workflow template. Customize it for your specific needs.

## How to Invoke

Run the workflow using:
```
workflow my-workflow
```

Or invoke it directly from the IDE command palette.

## Expected Inputs

- Customize this section based on what your workflow needs

## Generated Outputs

- Output file: `docs/my-workflow-output.md`

## Customization

1. Update `workflow.yaml` with your workflow name and description
2. Modify `instructions.md` to define your workflow steps
3. Customize `template.md` for your output format
4. Update `checklist.md` with validation criteria specific to your workflow

## Requirements

- BMAD core module installed
- Standard BMAD configuration in `.bmad/core/config.yaml`
